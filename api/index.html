Movie App RESTful API

User

Action Method Endpoint
Read a User GET api/user/read/${username}
Read all Users GET api/user/read/

Cinema

Action Method Endpoint
Read a Cinema GET api/cinema/read/${cinemaId}
Read all Cinemas GET api/cinema/read/
Create a Cinema POST api/cinema/
Update a Cinema PUT api/cinema/${cinemaId}
Delete a Cinema DELETE api/cinema/${cinemaId}

Movie

Action Method Endpoint
Read a Movie GET api/movie/read/${movieId}
Read all Movies GET api/movie/all/
Read all Owner's Movies GET api/movie/read/
Read all User's Favourite Movies GET api/movie/favourites/
Read all Movies played Today GET api/movie/today/
Search a Movie GET api/movie/search/${keyword}
Add a Movie as Fav/ Remove from Favs POST api/movie/favourite/
Create a Movie POST api/movie/
Update a Movie PUT api/movie/${movieId}
Delete a Movie DELETE api/movie/${movieId}
